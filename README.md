# Jelp-Backend
Backend logic for the Jelp project.

The Jelp Project is a website, much like Yelp, where users can read and create reviews for restaurants.

Technologies Utilized
---
- Spring Boot
- MySQL & JPA
- Mockito Testing
- Spring Security
